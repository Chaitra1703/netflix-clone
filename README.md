# 🎬 Netflix UI Clone

## 📌 About

Netflix UI Clone is a responsive front-end web application developed using **HTML, CSS, and JavaScript**. The project recreates the look and feel of the Netflix homepage with a modern layout, interactive elements, and a mobile-friendly design.


##  Live website link 
https://reliable-genie-fca4cd.netlify.app/

##  Repository

Chaitra1703/netflix-clone

## ✨ Features

* Responsive navigation bar
* Hero banner section
* Movie content rows
* Hover effects on movie cards
* FAQ accordion
* Responsive design for all screen sizes

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
